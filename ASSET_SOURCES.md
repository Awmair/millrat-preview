# Official asset sources

Every product, game and founder image used in this preview is hosted by BackerKit and surfaced on the current official [MILLRAT Pack! campaign landing page](https://www.backerkit.com/call_to_action/8aea9a55-41c0-4865-b148-535df295dd99/landing).

The source workspace did not contain downloadable MILLRAT asset files. To avoid recreating, modifying or misrepresenting official artwork, the preview references the official BackerKit-hosted files directly. These should be replaced with original files supplied by MILLRAT Studio before production launch.
